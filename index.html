The World Wide Web was created in the early nineties and has become a system supporting the sharing, storage, linking and displaying of specially formatted text documents using the markup language of HTML.

HTML is made up of, text, markup tags, and reference links to other pages and media (images, video, audio). Markup extends text into html making it richer.

The syntax of code is strict, and there are many rules, too many to memorize, if you forget something you can always look it up quickly w3.org website but if you obey the rules you can create amazing things!

Internet based HTML documents stored on servers can be displayed on a computer that has an internet connection by pointing a Web Browser to a unique internet address otherwise known as a URL. The rules for the tranfer and storage of these files from server to browser is called HTTP.

A HTML Element is made up of the content within a set of opening and closing tags.

  • Opening tag: All tags are contained within brackets i.e. the paragraph opening tag <p>.
  • Content: whatever you content happens to be.
  • Closing tag: the paragraph closing tag </p>.

    Note: some tags (void tags) do not require a closing tags e.g. the line break tag<br>.

      <opening tag><content><closing tag>

      <b><computer psychic hotline></b>

      Which in a browser looks like this:

      computer psychic hotline

    An Elements attributes are always specified at the start of a tag and provides additional info about the element.
    Example of a few tags:

    • For the anchor tag, the link addess is specified in the "href" attribute <a href="http://www.w3.org/image.jpg">This is a link</a>.
    • For the Image tag, a void tag, the name of the image source and the size of the image are all attributes <img src="image.jpg" alt="W3.com" width="104" height="142">
    .

    White Space and the difference between Block and Inline.

    In html, whitespace is always just one space, so to create space around elements/text use tags.
    Examples of Inline tags:<span> <br>. Inline Elements are normally displayed without line breaks.
    Examples of Block tags:<p> <div>. Block Elements are normally start with a new line.
      br tag <br> is used to insert a single line break in content <br> is a void tag so it doesn't need to be closed except for in xhtml where it shoud be closed like this: <br/>
      span tag <span> is used to group inline elements together for CSS styling, creates no change on its own More detail on span later ... it shoud be closed like this: </span>
      div tag <div> is used to group block elements together for CSS styling, by default browser always places a line break before and after the div element More detail on span later ... it shoud be closed like this: </div>
      p tag <p> by default browsers will automatically add a margin before and after the paragraph element</p>
